## 🎯 版本规划

### v2.0 - Aicy 升级版 🚧 开发中

#### Phase 1: 设计完成 ✅ (2025-12-24)

- [x] **好感度系统设计**
  - 9999点上限，100个关键时刻
  - 6-15天通关，能量密度高
  - 语义匹配判定（非关键词匹配）

- [x] **记忆系统架构**
  - Context构建方案
  - 数据库表结构设计
  - RAG检索设计

- [x] **人设 Prompt**
  - 六阶段人设（观察者→告别者）
  - 阶段1"破绽"机制
  - 亲密互动阶段指南

- [x] **世界观**
  - 2157年完整设定
  - EDIAN网络
  - Aicy背景故事

#### Phase 2: 技术实现 (进行中)

- [ ] **数据库**
  - PostgreSQL 建表
  - 对话/记忆/好感度存储

- [ ] **后端 API**
  - Node.js 服务器
  - Context 构建逻辑
  - 关键时刻触发检测

- [ ] **AI 集成**
  - DeepSeek API 对接
  - 人设 Prompt 注入

#### Phase 2: NSFW 内容 (Q1 2025)

- [ ] **解锁系统**
  - 好感度达标解锁
  - 渐进式内容开放

- [ ] **内容分级**
  - SFW 模式（默认）
  - NSFW 模式（需解锁）

- [ ] **安全机制**
  - 年龄验证
  - 内容警告

#### Phase 3: 视觉升级 (Q2 2025)

- [ ] **头像系统**
  - 动态表情
  - 情绪反应动画
  - 服装切换

- [ ] **UI 美化**
  - 更精致的界面
  - 动效增强
  - 主题切换

#### Phase 4: 币圈整合 (Q2 2025)

- [ ] **Token 机制**
  - 好感度用 Token 表示

- [ ] **Solana 整合**
